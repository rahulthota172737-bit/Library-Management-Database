\# 📚 Library Management Database



A relational database project built using MySQL to manage books, authors, members, staff, borrowing, reservations, and fines.



\## Features



\- Database normalization

\- Primary Keys

\- Foreign Keys

\- Constraints

\- Sample data

\- Book borrowing system

\- Reservation system

\- Fine management



\## Tables



\- Authors

\- Categories

\- Books

\- Members

\- Staff

\- Book\_Copies

\- Borrow\_Records

\- Reservations

\- Fines



\## Technologies Used



\- MySQL

\- SQL
